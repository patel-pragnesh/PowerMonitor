# PowerMonitor
PowerMonitor is an Open Source software currently in development.
It aims at allowing the users to control and monitor their electrical consumption.
The solution is composed of 4 main components:
* The master module
* The slave module
* The distant server
* 4 user interfaces
  * An Android app
  * An iOS app
  * A Windows Phone app
  * A Web app
